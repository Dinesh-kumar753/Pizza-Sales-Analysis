# 🍕 Pizza Sales Analysis using SQL

## 📌 Project Overview

This project analyzes pizza sales data to uncover key business insights such as revenue trends, customer preferences, and ordering behavior.

The analysis is performed using SQL by working with multiple related tables and applying advanced querying techniques.

---

## 🎯 Objective

* Analyze sales performance
* Identify top-selling pizzas
* Understand customer ordering patterns
* Discover revenue drivers

---

## 🗂️ Dataset

The dataset consists of the following tables:

* `orders`
* `order_details`
* `pizzas`
* `pizza_types`

---

## 🛠️ Skills Demonstrated

* SQL Joins
* Aggregations (SUM, COUNT, AVG)
* GROUP BY & ORDER BY
* Window Functions
* Business Data Analysis

---

## 📊 Key Analysis Performed

* Total number of orders placed
* Total revenue generated
* Highest-priced pizza
* Most common pizza size
* Top 5 most ordered pizza types
* Category-wise sales analysis
* Hourly order distribution
* Average pizzas ordered per day
* Top pizzas based on revenue
* Revenue contribution (%) by pizza type
* Cumulative revenue over time
* Category-wise top-performing pizzas using ranking

---

## 📈 Key Insights

* Identified peak order hours and high-demand pizza sizes
* Found top revenue-generating pizza categories
* Analyzed contribution of each pizza to total revenue
* Observed sales trends over time using cumulative revenue
